# HTML

  • 页面导入样式时，使用link和@import有什么区别？

    1. link是HTML标签，@import是css提供的。

    2. link没有兼容性问题，@import在ie6及以上才兼容。

    3. link可以通过js操作DOM来引入不同的样式表，而@import不可以。

（注意：如果@import在link引入的样式表里面，则必须放在头部，否则不会生效；且@import引入的样式表会在页面加载完成后加载，但同选择器的权重会比link里的低。）

# CSS

  • 圣杯布局和双飞翼布局的理解和区别，并用代码实现。

作用：圣杯布局和双飞翼布局解决的问题是一样的，主要是为了解决中间自适应，两边定宽的三栏布局方式，并且中间的区域按文档流的方式优先加载。(注意：务必理解浮动和负外边距的原理)

浮动的特性：浮动框的上边缘会去贴当前行盒的上边缘或是之前浮动框的下边缘，左浮动框的左边缘会去贴包含框的左边缘，或者他之前的左浮动框的右边缘。如果当前行剩余的空间容不下一个浮动框，他就会换行。（右浮动同理）。

圣杯布局：

<!DOCTYPE html>
<html lang="zh-CN">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>圣杯布局</title>
</head>
<style>
    body {
        min-width: 900px;//左边区块的宽度*2+右边区块的宽度
        font-weight: bold;
        font-size: 20px;
    }

    #header,
    #footer {
        background: rgba(29, 27, 27, 0.726);
        text-align: center;
        height: 60px;
        line-height: 60px;
    }

    #container {
        padding-left: 200px;
        padding-right: 350px;
        overflow: hidden;
    }

    #container .column {
        position: relative;
        float: left;
        text-align: center;
    }

    #center {
        width: 100%;
        background: rgb(206, 201, 201);
    }

    #left {
        width: 200px;
        right: 200px;
        margin-left: -100%;//不能小于左边区块的宽度，否则布局会错乱，所以body要设置最小宽度
        background: rgba(95, 179, 235, 0.972);
    }

    #right {
        width: 350px;
        margin-right: -350px;
        background: rgb(231, 105, 2);
    }
</style>

<body>
    <div id="header">头部</div>
    <div id="container">
        <div id="center" class="column">中间</div>
        <div id="left" class="column">左边</div>
        <div id="right" class="column">右边</div>
    </div>
    <div id="footer">底部</div>
</body>

</html>

# javascript

这是一道大题目，把考点拆成了4个小项；需要侯选人用递归算法实现（限制15行代码以内实现；限制时间10分钟内完成）： a) 生成一个长度为5的空数组arr。 b) 生成一个（2－32）之间的随机整数rand。 c) 把随机数rand插入到数组arr内，如果数组arr内已存在与rand相同的数字，则重新生成随机数rand并插入到arr内[需要使用递归实现，不能使用for/while等循环] d) 最终输出一个长度为5，且内容不重复的数组arr。

let myRandDom = (start, end) => Math.floor(Math.random() * (end - start + 1)) + start;
function createArr(arr = new Array(5), index = 0, randomNum = myRandDom(2, 32)) {
    !arr.includes(randomNum) && (arr[index++] = randomNum);
    index != arr.length && createArr(arr, index);
    return arr;
}
